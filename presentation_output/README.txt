BitClock Presentation Output
============================

This folder contains the auto-generated presentation materials for the BitClock project.

Contents:
---------
1. presentation.html
   - INTERACTIVE SLIDES. Open this file in Chrome, Edge, or Firefox.
   - Use Left/Right arrow keys to navigate.
   - Click the 'Memo' icon in the top right to toggle speaker notes.
   - To convert to PDF: Open in browser -> Print -> Save as PDF (Select "Background Graphics" in print settings).

2. slides.json
   - Structured data representing the 14 slides. Useful for programmatic processing.

3. speaker_notes.md
   - Contains the script/notes for the presenter for each slide.

4. assets/
   - Contains all the generated SVG diagrams (Architecture, Gantt, Wireframes, etc.).

Note on PPTX:
-------------
Due to environment restrictions on generating binary files, an HTML-based presentation engine was created instead of a .pptx file. The HTML file provides a polished, full-screen experience similar to PowerPoint.
